# DropZo 🚰

A smart doorstep delivery management Android application built to streamline 
order placement, real-time tracking, and admin operations.

## Features
- 📦 Customer order placement
- 🔄 Real-time order status updates
- 🛠️ Admin dashboard – manage & monitor all deliveries
- 📡 Live cross-device sync via Firebase
- ⚡ Fast, lightweight & offline-resilient

## Tech Stack
- Android Studio
- Java
- Firebase Realtime Database
